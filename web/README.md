# Shot Sync Web Version

Browser-based basketball shot form analyzer using MediaPipe.js.

## Features

- Real-time pose detection
- Shot form comparison
- Visual analysis charts
- Instant feedback

## How to Use

1. Record a benchmark shot
2. Record your shot
3. Get instant analysis and feedback

## Technical Details

- **MediaPipe.js** - Pose detection
- **Chart.js** - Data visualization
- **100% Client-side** - No server needed
